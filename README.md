# LockPickingLookup

Have you ever been stuck picking a lock and wished you could pull up a LockPickingLawyer or BosnianBill video immediately to help?

That's all this site is; a simple search across more than 3000 videos without any funky algorithms, adverts or random videos from other people.

The results are ordered by how close they match your search, and if you're getting too many you can prefix words with a hyphen to exclude them, like this: 'Master Lock -Combination'. The search will return the closest 100 matches.

The LockPickingLawyer and BosnianBill have no connection with this site, it's just a search front-end to their publicly-available YouTube playlists.

This project is created purely for fun and out of respect for these titans' contributions to locksport and security education.
